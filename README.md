
# Contactless Automatic Switching using LPC2148

This project implements a **contactless automatic switching system** using an **LPC2148 ARM7 microcontroller**, IR sensors, relay module, and a DC motor driver.  
The system enables **hands-free control** of electrical devices, improving hygiene, safety, and convenience.

Contactless switching eliminates the need for physical interaction with switches.  
An **IR sensor** detects the presence of a hand or object and sends a signal to the **LPC2148 microcontroller**, which processes the input and activates a **relay** and **DC motor** accordingly.
